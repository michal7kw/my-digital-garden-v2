# Antinuclear Antibodies Titer

## Overview
an-tee-noo-klee-ur an-tih-bod-eez tie-ter An antinuclear antibodies (ANA) titer measures the concentration of autoantibodies present linked to an autoimmune condition. Higher titers can signal more autoimmune activity, but does not confirm the presence of an autoimmune condition.This test is only done if ANA screen is positive.

**Evidence Level:** ⭐⭐⭐ (3/5)
