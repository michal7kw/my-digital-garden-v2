<h1>Memory and Learning</h1>
<h2>Overview</h2>
<p>Memory is the faculty by which the brain encodes, stores, and retrieves information and past experiences. It is not a single entity but a collection of distinct systems served by different brain circuits. The hippocampus is the gateway for new declarative (explicit) memories — encoding both personal events (episodic memory) and factual knowledge (semantic memory). Working memory, the ability to hold and manipulate information in real time, depends on the prefrontal cortex. Procedural memory (skills and habits) relies on the basal ganglia and cerebellum. Learning — the process of acquiring new information or modifying existing knowledge — depends on synaptic plasticity, particularly long-term potentiation (LTP) in hippocampal circuits.</p>
<h2>Memory Types</h2>
<ul>
<li><strong>Episodic memory</strong> — Personal events with spatiotemporal context (&quot;what/where/when&quot;) — hippocampus</li>
<li><strong>Semantic memory</strong> — Factual knowledge and concepts — anterior temporal lobe, neocortex</li>
<li><strong>Working memory</strong> — Active maintenance and manipulation of information — dorsolateral PFC, parietal cortex</li>
<li><strong>Procedural memory</strong> — Motor skills and habits — basal ganglia, cerebellum</li>
<li><strong>Spatial memory</strong> — Environmental layouts and navigation — hippocampus, entorhinal cortex</li>
</ul>
<h2>Key Mechanisms</h2>
<ul>
<li><strong>Long-term potentiation (LTP)</strong> — Persistent strengthening of synapses through repeated activation (NMDA-dependent)</li>
<li><strong>Memory consolidation</strong> — Transfer from hippocampus to neocortex during sleep (especially slow-wave sleep)</li>
<li><strong>Reconsolidation</strong> — Reactivated memories become labile and can be updated or weakened</li>
<li><strong>Neurogenesis</strong> — New hippocampal neurons (dentate gyrus) contribute to pattern separation and new memory formation</li>
</ul>
<h2>Neurotransmitter Modulation</h2>
<ul>
<li><strong>Acetylcholine</strong> — Basal forebrain cholinergic input enhances encoding strength; target of Alzheimer's drugs</li>
<li><strong>Glutamate</strong> — NMDA receptors are required for LTP and new memory formation</li>
<li><strong>Dopamine</strong> — Novelty signals from VTA enhance hippocampal encoding (&quot;memory reward tag&quot;)</li>
<li><strong>Norepinephrine</strong> — Emotional arousal enhances memory consolidation via amygdala-hippocampus interaction</li>
</ul>
<h2>Clinical Relevance</h2>
<ul>
<li><strong>Alzheimer's disease</strong> — Progressive episodic memory loss (hippocampal and entorhinal atrophy)</li>
<li><strong>Mild cognitive impairment</strong> — Transitional stage with measurable memory decline</li>
<li><strong>Age-related memory decline</strong> — Normal aging reduces encoding efficiency and retrieval speed</li>
<li><strong>Traumatic brain injury</strong> — Disrupted memory consolidation (diffuse axonal injury)</li>
<li><strong>PTSD</strong> — Aberrant emotional memory consolidation and impaired fear extinction</li>
</ul>
