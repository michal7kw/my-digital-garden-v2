<h1>Executive Function</h1>
<h2>Overview</h2>
<p>Executive functions (EFs) are the top-down mental processes needed when going on &quot;autopilot&quot; would be ill-advised or impossible. Adele Diamond's influential framework identifies three core EFs: inhibitory control (resisting impulses and distractions), working memory (holding and manipulating information in mind), and cognitive flexibility (switching between perspectives or tasks). These core EFs give rise to higher-order EFs including reasoning, problem-solving, and planning. The prefrontal cortex is the primary neural substrate, and EFs follow an inverted-U function for both dopamine and norepinephrine — too little (ADHD, Parkinson's) or too much (acute stress, psychosis) impairs performance.</p>
<h2>Core Components</h2>
<ul>
<li><strong>Working memory</strong> — Holding and manipulating information in mind (dorsolateral PFC, parietal cortex)</li>
<li><strong>Cognitive flexibility</strong> — Switching between tasks or mental sets (ventrolateral PFC, basal ganglia)</li>
<li><strong>Inhibitory control</strong> — Suppressing prepotent responses and resisting distractions (right inferior frontal gyrus, subthalamic nucleus)</li>
<li><strong>Planning</strong> — Generating and evaluating multi-step action sequences (dorsolateral PFC, frontopolar cortex)</li>
<li><strong>Decision-making</strong> — Evaluating options under uncertainty (orbitofrontal cortex, ventromedial PFC)</li>
</ul>
<h2>Neurotransmitter Modulation</h2>
<ul>
<li><strong>Dopamine</strong> — D1 receptor stimulation in PFC enhances working memory and cognitive flexibility; inverted-U dose-response</li>
<li><strong>Norepinephrine</strong> — Alpha-2A adrenergic stimulation in PFC strengthens &quot;signal&quot; in working memory networks</li>
<li><strong>Glutamate</strong> — NMDA receptors on PFC pyramidal neurons are essential for sustained activity (working memory)</li>
<li><strong>GABA</strong> — Interneuron-mediated inhibition shapes the tuning of PFC networks</li>
</ul>
<h2>Clinical Relevance</h2>
<ul>
<li><strong>ADHD</strong> — Core deficit in inhibitory control and working memory (prefrontal dopamine hypofunction)</li>
<li><strong>Frontal lobe injury</strong> — Impaired planning, disinhibition, personality change</li>
<li><strong>Schizophrenia</strong> — Working memory deficits correlate with functional outcome (DLPFC hypoactivation)</li>
<li><strong>Depression</strong> — Executive dysfunction predicts treatment resistance and functional impairment</li>
<li><strong>Parkinson's disease</strong> — Frontostriatal dopamine depletion causes executive rigidity</li>
</ul>
