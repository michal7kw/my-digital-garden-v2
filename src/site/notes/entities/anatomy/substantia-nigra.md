<h1>Substantia Nigra</h1>
<h2>Description</h2>
<p>Midbrain structure containing the brain's densest dopamine-producing neurons. The pars compacta provides dopaminergic input to the striatum; its degeneration causes Parkinson's disease.</p>
<h3>Subdivisions</h3>
<ul>
<li><strong>Pars compacta (SNpc)</strong> — Dopaminergic neurons, nigrostriatal pathway origin</li>
<li><strong>Pars reticulata (SNpr)</strong> — GABAergic output, basal ganglia circuit</li>
</ul>
