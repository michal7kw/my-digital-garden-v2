<h1>Hippocampus</h1>
<h2>Description</h2>
<p>Seahorse-shaped structure in the medial temporal lobe, essential for episodic memory formation, spatial navigation, and one of the few brain regions where adult neurogenesis occurs.</p>
<h3>Subfields</h3>
<ul>
<li><strong>CA1</strong> — Memory consolidation, output to cortex</li>
<li><strong>CA3</strong> — Pattern completion, autoassociative network</li>
<li><strong>Dentate gyrus</strong> — Pattern separation, adult neurogenesis</li>
<li><strong>Subiculum</strong> — Output relay to hypothalamus and cortex</li>
</ul>
