<h1>Entorhinal Cortex</h1>
<h2>Description</h2>
<p>The entorhinal cortex (EC) is a region in the medial temporal lobe that serves as the major interface between the hippocampus and the neocortex. It is the primary input to the hippocampus via the perforant pathway and the primary output relay back to cortical association areas. The EC contains the grid cells discovered by May-Britt and Edvard Moser (2014 Nobel Prize), which provide a metric spatial coordinate system for navigation. It also processes olfactory information and is the earliest cortical site of tau pathology in Alzheimer's disease.</p>
<h3>Key Features</h3>
<ul>
<li><strong>Perforant path</strong> — The main cortical input to the hippocampus, essential for memory encoding</li>
<li><strong>Grid cells</strong> — Neurons with hexagonal spatial firing patterns providing a coordinate system</li>
<li><strong>Layer II</strong> — Projects to hippocampal dentate gyrus and CA3 (episodic memory input)</li>
<li><strong>Layer III</strong> — Projects to hippocampal CA1 (temporal context)</li>
<li><strong>Layer V</strong> — Receives hippocampal output and relays to neocortex</li>
</ul>
