<h1>Acetylcholine</h1>
<h2>Overview</h2>
<p>First neurotransmitter discovered. Critical for memory, learning, attention, and neuromuscular function. The basal forebrain cholinergic system is a primary target in Alzheimer's disease.</p>
<p><strong>Chemical Formula:</strong> C7H16NO2+<br />
<strong>Molecular Weight:</strong> 146.21</p>
<h2>Primary Functions</h2>
<ul>
<li>Memory and learning</li>
<li>Attention and arousal</li>
<li>Neuromuscular transmission</li>
<li>Parasympathetic nervous system</li>
<li>REM sleep</li>
</ul>
<h2>Receptor Types</h2>
<ul>
<li>Nicotinic (nAChR) — ion channel, fast excitatory</li>
<li>Muscarinic M1 — cortical, learning (Gq)</li>
<li>Muscarinic M2 — cardiac, autoreceptor (Gi)</li>
<li>Muscarinic M3 — smooth muscle, glands (Gq)</li>
<li>Muscarinic M4 — striatal (Gi)</li>
<li>Muscarinic M5 — dopamine modulation (Gq)</li>
</ul>
<h2>Synthesis Pathway</h2>
<p>Choline + Acetyl-CoA → Acetylcholine (choline acetyltransferase / ChAT)</p>
<h2>Associated Conditions</h2>
<ul>
<li>Alzheimer's disease (cholinergic neuron loss in basal forebrain)</li>
<li>Myasthenia gravis (nAChR antibodies)</li>
<li>Lewy body dementia (cholinergic deficit)</li>
<li>Parkinson's disease dementia (dual cholinergic/dopaminergic loss)</li>
</ul>
