# Edema

## Description

Abnormal accumulation of fluid in tissues causing swelling.

## üß™ Associated Biomarkers

The following biomarkers should be evaluated when investigating edema:

- [[Albumin]] - Low albumin causes fluid to leak from blood vessels into tissues
- [[Creatinine]] - Elevated creatinine indicates kidney dysfunction which can cause edema
