# GABA

## Overview
Gamma-aminobutyric acid, the brain's primary inhibitory neurotransmitter. GABA reduces neuronal excitability, promotes relaxation, reduces anxiety, and facilitates sleep. It provides the "brake" to glutamate's "accelerator."
