# Modafinil

## Overview
Wakefulness-promoting agent used for narcolepsy, sleep apnea, and shift work disorder. Popular off-label as a cognitive enhancer. Mechanism distinct from traditional stimulants.

**Short Description**: Wakefulness-promoting agent for sleep disorders

## Pharmacological Information

**DrugBank ID**: DB00745

**Drug Class**: Eugeroic (Wakefulness-Promoting Agent)

**Synonyms**:
- Provigil
- Alertec
- Modavigil

### Mechanism of Action
Promotes wakefulness via DAT inhibition and multiple neurotransmitter pathways
