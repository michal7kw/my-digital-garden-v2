<h1>Immune System Disorder</h1>
<h2>Overview</h2>
<p>Umbrella term for conditions where the immune system functions abnormally. Includes immunodeficiency (underactive — frequent infections), autoimmunity (overactive against self — RA, lupus, MS, T1DM), allergic/atopic disorders (overactive against harmless antigens), and autoinflammatory conditions. The immune system has innate (fast, non-specific: neutrophils, macrophages, NK cells) and adaptive (slow, specific: T cells, B cells, antibodies) arms. Both can malfunction.</p>
<h2>Symptoms</h2>
<ul>
<li><strong>Immunodeficiency</strong>: Recurrent infections, slow wound healing, chronic fatigue</li>
<li><strong>Autoimmune</strong>: Depends on target organ — joint pain (RA), skin rashes (lupus), thyroid dysfunction (Hashimoto's), gut symptoms (celiac)</li>
<li><strong>Allergic</strong>: Rhinitis, urticaria, asthma, anaphylaxis</li>
<li><strong>General</strong>: Chronic fatigue, systemic inflammation, poor stress tolerance</li>
</ul>
<h2>Treatment Approaches</h2>
<ul>
<li><strong>Immunodeficiency</strong>: Treat underlying cause, immunoglobulin replacement (IVIG/SCIG), prophylactic antibiotics</li>
<li><strong>Autoimmune</strong>: Immunosuppression (corticosteroids, methotrexate, biologics), disease-specific DMARDs</li>
<li><strong>Allergic</strong>: Antihistamines, immunotherapy (desensitization), biologics (omalizumab, dupilumab)</li>
<li><strong>Foundation for all</strong>: Anti-inflammatory diet, adequate sleep (7-9h), regular exercise, stress management, vitamin D optimization</li>
<li><strong>Supplements</strong>: Vitamin D (2000-5000 IU — immune modulation, reduces autoimmune incidence per VITAL trial), vitamin C (immune cell function), zinc (thymulin cofactor), probiotics (gut-immune axis), NAC (glutathione precursor)</li>
</ul>
