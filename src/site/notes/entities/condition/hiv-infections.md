<h1>HIV Infections</h1>
<h2>Overview</h2>
<p>A chronic viral infection that attacks the body's immune system, specifically CD4 T cells, potentially leading to AIDS if untreated.</p>
<h2>Symptoms</h2>
<ul>
<li>Acute: Fever, lymphadenopathy, pharyngitis, rash (2-4 weeks post-exposure)</li>
<li>AIDS (CD4 &lt;200): Opportunistic infections (PCP, CMV, toxoplasmosis)</li>
</ul>
<h2>Treatment Approaches</h2>
<ul>
<li><strong>ART (start immediately)</strong>: Integrase inhibitor-based regimens (bictegravir/emtricitabine/TAF)</li>
<li><strong>Long-acting</strong>: Cabotegravir + rilpivirine injections (every 2 months)</li>
<li><strong>Goal</strong>: Undetectable viral load = untransmittable (U=U)</li>
<li><strong>PrEP</strong>: Tenofovir/emtricitabine or cabotegravir for prevention</li>
<li><strong>Supplements</strong>: Vitamin D, probiotics, NAC, selenium</li>
</ul>
